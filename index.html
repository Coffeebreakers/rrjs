rr.popupMessage:

  • Simples (padrão) Exemplo - código
    rr.popupMessage('Demonstração rr.popupMessage');
  • Título Exemplo - código
    rr.popupMessage('Demonstração rr.popupMessage<br />com Título',{title: 'rr.popupMessage'});
  • Largura Variável Exemplo - código
    rr.popupMessage('Demonstração rr.popupMessage<br />com largura variavel',{width: '500'});
  • Altura Variável Exemplo - código
    rr.popupMessage('Demonstração rr.popupMessage<br />com altura variavel',{height: '300'});
  • Título, Altura e Largura Variáveis Exemplo - código
    rr.popupMessage('Demonstração rr.popupMessage<br />com altura e largura variaveis<br />e titulo',{title: 'rr.popupMessage', width: '500', height: '300'});
  • Classe Auxiliar Exemplo - código
    rr.popupMessage('Demonstração rr.popupMessage<br />com classe adicional',{auxClass: 'rrPopupMessageAux'});
  • com função de Callback Exemplo - código
    rr.popupMessage('Demonstração rr.popupMessage<br />com função de Callback', { okFunction: demo.okFunction } );
